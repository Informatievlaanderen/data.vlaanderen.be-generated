Dit document beschrijft een implementatiemodel, in dit geval het Wegenregister. Een implementatiemodel is een toepassing van vocabularia voor een concreet doel. Het beschrijft in alle detail (zoals kardinaliteiten, codelijsten, datatypes, ...) de gewenste datastructuur.

## Samenvatting {#summary} Dit is een implementatiemodel van het applicatieprofiel [wegenregister](https://data.vlaanderen.be/doc/applicatieprofiel/wegenregister). De applicatie waarop dit profiel betrekking heeft is het Vlaams wegenregister. ## Status van dit document {#status} Dit implementatiemodel heeft status [Kandidaat-standaard](https://data.vlaanderen.be/id/concept/StandaardStatus/Kandidaat-standaard) en werd uitgegeven op 2022-12-02. Informatie over het gevolgde proces en de beslissingen om tot deze specificatie te komen zijn beschikbaar in het [standaardenregister](https://data.vlaanderen.be/standaarden/kandidaat-standaarden/applicatieprofiel-wegenregister/). ## Licentie {#license} Deze specificatie van [Digitaal Vlaanderen][] is gepubliceerd onder de ["Modellicentie Gratis Hergebruik - v1.0"][Licentie]. ## Conformiteit {#comformence} De conformiteit voor implementatiemodellen is [hier][Comformiteit] te vinden. ## Overzicht {#overview} In dit document wordt correct gebruik van de volgende entiteiten toegelicht:
| Aantal rijstroken | Europees wegsegment | Genummerd wegsegment | Nationaal wegsegment | Ongelijkgrondse kruising | Straatnaam | Wegbreedte | Wegenregister organisatie | Wegknoop | Wegsegment | Wegverharding | [![](/assets/oslo/doc/implementatiemodel/wegenregister/ontwerpstandaard/2022-08-30/html/overview.jpg)](/assets/oslo/doc/implementatiemodel/wegenregister/ontwerpstandaard/2022-08-30/html/overview.jpg) ## Entiteiten

Aantal rijstroken

**Beschrijving**
Het aantal rijstroken van een wegsegment. **Subklasse van**
WegenregisterObject **Eigenschappen**
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: aantal, richting, tot positie, van positie. | Eigenschap | Verwacht Type | Kardinaliteit | Beschrijving | Gebruik | Codelijst | |------------|---------------|---------------|--------------|---------|-----------| | aantal | | 1 | Het aantal door (doorlopende of onderbroken) witte strepen begrensde delen van de rijbaan, plaats biedend aan autoverkeer. | | | | richting | Richtingwaarde aantal rijstroken | 1 | De richting waarin het aantal rijstroken geldt t.o.v. de zin van de wegsegmentvector. | | | | tot positie | | 1 | De eindpositie, uitgedrukt als de afstand langs de veelhoekslijn vanaf de start van het wegsegment. | | | | van positie | | 1 | De startpositie, uitgedrukt als de afstand langs de veelhoekslijn vanaf de start van het wegsegment. | | |

Europees wegsegment

**Beschrijving**
Een Europese weg maakt deel uit van het internationale E-wegennetwerk, gekenmerkt door zijn Europese routenummer. **Subklasse van**
WegenregisterObject **Eigenschappen**
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: Europees wegnummer. | Eigenschap | Verwacht Type | Kardinaliteit | Beschrijving | Gebruik | Codelijst | |------------|---------------|---------------|--------------|---------|-----------| | Europees wegnummer | String | 1 | Code die de route in het internationaal E-wegennetwerk identificeert. De code begint steeds met de letter E en wordt vervolledigd door een getal met 1,2 of 3 cijfers. | | |

Genummerd wegsegment

**Beschrijving**
Een genummerde weg is een weg die in een bepaald land van bovenlokaal belang is of door een hogere overheidsinstantie wordt beheerd, en steeds genummerd wordt. **Subklasse van**
WegenregisterObject **Eigenschappen**
Voor deze entiteit zijn de volgende eigenschappen gedefinieerd: ident8, richting, volgnummer. | Eigenschap | Verwacht Type | Kardinaliteit | Beschrijving | Gebruik | Codelijst | |------------|---------------|---------------|--------------|---------|-----------| | ident8 | String | 1 | Technische code van een genummerde weg toegekend door het Agentschap Wegen en Verkeer. | | | | richting | Richtingwaarde genummerde weg | 1 | De richting waarin de ident 8 geldt t.o.v. de zin van de wegsegmentvector. | | | | volgnummer | String | 1 | Volgnummer van het Wegsegment in de ordening van de genummerde weg. | | |

Straatnaam

**Beschrijving**
Naam die officieel werd toegekend ofwel aan een straat (baan, doorgang, plein), ofwel aan een gehucht en waaraan 'huisnummers' kunnen zijn gekoppeld. **Eigenschappen**
Voor deze entiteit zijn geen eigenschappen gedefinieerd.
