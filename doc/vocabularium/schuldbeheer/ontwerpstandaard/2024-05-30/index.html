Afhankelijkheden

Dit document beschrijft een vocabularium, in dit geval OSLO Schuldbeheer. Een vocabularium is een verzameling van herbruikbare termen, namelijk klassen en eigenschappen, waarvan hun betekenis is vastgelegd door middel van een label en definitie. Voor termen gedefinieerd in dit domein zijn er nog bijkomende formele afspraken beschikbaar. Voor externe termen worden de formele afspraken overgenomen van de externe bron.

Status van dit document

1. Licentie

Deze specificatie van Informatie Vlaanderen is gepubliceerd onder de "Modellicentie Gratis Hergebruik - v1.0".

2. Conformiteit

Een uitwisseling van gegevens, op welke manier deze uitwisseling ook gebeurt, is conform aan dit vocabularium wanneer het de terminologie (klassen en eigenschappen) gebruikt op een manier die consistent is met de semantiek zoals opgesteld in de nieuwste versie van de specificatie (domein, bereik, definitie en gebruik) en het geen terminologie gebruikt uit andere vocabularia als alternatief voor de voorgestelde terminologie opgenomen in dit vocabularium.

4. Klassen

Deze sectie geeft een formele definitie aan elke klasse.

Klasse Aanmaning

Type Klasse
URI https://data.vlaanderen.be/ns/schuldbeheer#Aanmaning
Specialisatie van
  • http://www.w3.org/ns/prov#Activity
Definitie Schriftelijk berichtgeving van een schuldeiser naar een schuldenaar om deze schuldenaar te herinneren aan een vordering die niet is voldaan terwijl de vervaldatum van deze vordering reeds verstreken is.

Type Klasse
URI https://data.vlaanderen.be/ns/schuldbeheer#Afbetalingsplan
Definitie Een gestructureerd schema dat wordt gebruikt om een schuld af te lossen.

Klasse Vordering

Type Klasse
URI https://data.vlaanderen.be/ns/schuldbeheer#Vordering
Specialisatie van
  • http://www.w3.org/ns/prov#Activity
Definitie Aanspraak van een partij (de schuldeiser) op een bepaalde prestatie vanwege een andere partij (de schuldenaar).
