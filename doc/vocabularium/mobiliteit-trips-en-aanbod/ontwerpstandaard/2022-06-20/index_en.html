This document describes a vocabularium, for this case Mobility: Trips & Offers (Vocabulary). Please note that this vocabularium is an English translation of the Dutch vocabularium OSLO-Mobiliteit: Trips & Aanbod. A vocabularium is a collection of reusable terms (classes and properties), whose meaning is given by a label and a definition. There are additional formal agreements available for the terms defined in this domain. The additional formal agreements can be found in external sources for external terms.

Summary

The vocabulary "OSLO-Mobility: Trips and Offers" defines terms that apply to (1) trips undertaken by individuals and (2) the mobility services available to them.

The trips are defined in terms of the route taken, the services/resources/infrastructure used, the operator, the tickets purchased, traveller preferences and characteristics.

The mobility services include concepts such as the resources to be reserved by the traveller, the infrastructure available to the service, information about the provider and operator, and the licences they hold.

4. Classes

This section defines each class in a formal manner.

Klasse Accessibility

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Toegankelijkheid
Definition Degree to which something can be accessed.

Usage

 E.g. physical accessibility of a service or building for persons with disabilities.

Klasse Availability

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Beschikbaarheid
Definition Degree to which something is available for use.

Usage

 E.g. hours when a service is offered or opening hours of a building.

Klasse Booking

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Boeking
Definition Committing to a Trip by purchasing a Ticket and/or reserving a Resource.

Usage

 E.g. buying a train ticket or reserving a seat on the train.

Klasse Characteristic

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Kenmerk
Definition Feature of the Traveller.

Usage

 E.g. age, disability, driving licence.

Klasse Executor

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Uitvoerder
Narrower definition for
  • http://purl.org/dc/terms/Agent
Definition Agent that executes the Mobility Service.

Klasse License

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Licentie
Definition Formal authorisation to offer a Mobility Service.

Klasse MobilityService

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Mobiliteitsdienst
Definition Service that enables movement of Persons between Locations.

Usage

 This includes classical mobility services such as public transport (typically offered as a scheduled service, e.g. Metro Line 5 in Brussels), but also partial transport such as free-floating partial steps (e.g. partial steps in large cities).

Klasse Preference

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Voorkeur
Definition Preference of the Traveller.

Usage

 E.g. type of route, means of transport ...

Klasse Provider

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Aanbieder
Narrower definition for
  • http://www.w3.org/ns/regorg#RegisteredOrganization
Definition Registered Organisation providing Mobility Services.

Usage

 By a Registered Organisation, we mean an organisation with a legal status (established through registration), e.g. SA, NPO...

Klasse Pseudonym

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Pseudoniem
Definition Name of a Person differing from his real name.

Klasse Reservation

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Reservatie
Definition Characteristics of the Reservation of a Resource at a Reservation.

Klasse Resource

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Resource
Definition Means the Mobility Service can offer to the Passenger to make the trip.

Usage

 E.g. a seat on a train, a shared bicycle at a station, a parking space in a car park.

Klasse Route

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Route
Definition Possible, chosen or travelled route between two Locations (respectively the departure point and the arrival point).

Klasse RouteDescription

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Routebeschrijving
Definition Description of a Route

Usage

 Typically the description as it would be shown in a Route Planner.

Klasse RouteNode

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Routeknoop
Definition Significant position in the Route that always coincides with the beginning or the end of a Route segment.

Klasse RoutePlanner

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Routeplanner
Narrower definition for
  • http://purl.org/dc/terms/Agent

Klasse RouteSegment

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Routesegment
Narrower definition for
  • https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Route
Definition Part of a Route that is travelled without a transfer using the same means of transport.e

Klasse Timetable

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Dienstregeling
Definition Planning in time and space of a service.

Klasse Transfer

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Overstap
Definition Logical connection between two Route nodes.

Usage

 Means to describe the transition between two Route segments if not represented by a Route segment itself. One can interchange between two same Route nodes (e.g. changing from one train to another at the same station) or between two different ones (e.g. interchanging from the station car park to the station platforms).

Klasse TransportConnection

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Transportconnectie
Narrower definition for
  • https://data.vlaanderen.be/ns/netwerk#Connectie
  • https://data.vlaanderen.be/ns/transportnetwerk#Transportobject
Definition Logical connection between two or more Transport objects in different Transport Networks.

Klasse TransportTicket

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Vervoersbewijs
Definition Proof of certain right to transport or services associated with it.

Usage

 Transport or service to be consumed or already consumed. Typically a ticket or a subscription to a mobility service such as public transport, but can also be a parking ticket, for example.

Klasse Traveller

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Reiziger
Narrower definition for
  • http://purl.org/dc/terms/Agent
Definition Person who undertakes or wants to undertake a Trip.

Klasse Value

Type Class
URI https://data.vlaanderen.be/ns/mobiliteit/trips-en-aanbod#Waarde
Definition The concrete, quantitative or qualitative interpretation of a quantity or category.

Usage

 Quantitative e.g. weight, qualitative e.g. category.
