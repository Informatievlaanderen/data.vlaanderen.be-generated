Este documento describe un vocabularium, para este caso Infraestructura (vocabulario). Un vocabulario es una colección de términos reutilizables (clases y propiedades), cuyo significado viene dado por una etiqueta y una definición. Hay acuerdos formales adicionales disponibles para los términos definidos en este dominio. Los acuerdos formales adicionales se pueden encontrar en fuentes externas para términos externos.

Resumen

El vocabulario define una serie de entidades y propiedades temáticas además del modelo de datos conceptuales de dominio público.

Este documento es una especificación oficial de un vocabulario y proporciona un contexto compartido de términos para algunos conceptos. Esta especificación sirve para aclarar la definición, el uso, el dominio y el rango de los términos RDF dentro de este dominio. Los términos definidos en este vocabulario son persistentes y, como consecuencia, no cambiarán excepto para aclarar aún más la definición existente. Los términos pueden marcarse como obsoletos y reemplazarse en versiones más recientes de esta especificación.
